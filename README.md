# Drake Portfolio

Dark-themed responsive portfolio (Framer Designer & Developer) — Next.js, Tailwind CSS, Three.js particle background, profile card, and section-based layout.

---

## Screenshots & design

**[→ Design documentation (DESIGN.md)](./DESIGN.md)** — Layout, colors, components, and **all project screenshots** (mobile + desktop). Images are in `public/design-docs/` and display in that file on GitHub.

---

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

---

## Stack

- **Next.js** (App Router)
- **Tailwind CSS**
- **Three.js** (particle background)
- **TypeScript**
